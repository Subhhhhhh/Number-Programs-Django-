PROJECT OVERVIEW
---------------------------------------------------------------
This is a Django-based web application that allows users to test and analyze
various mathematical number properties. The user can input a number and check
multiple number theory concepts with instant results on the same page using a
simple and clean Bootstrap interface.

The application contains multiple cards where each performs a different number
property check or conversion.

INSTALLATION & SETUP INSTRUCTIONS
---------------------------------------------------------------
1. Clone the repository
   git clone https://github.com/Subhhhhhh/Number-Programs-Django-.git

2. Move to the project directory
   cd Number-Programs-Django-

3. Create a virtual environment
   python -m venv venv

4. Activate the environment
   venv\Scripts\activate         (Windows)
   source venv/bin/activate      (Linux / Mac)

5. Install required dependencies
   pip install django

6. Apply database migrations
   python manage.py migrate

7. Run the development server
   python manage.py runserver

8. Open browser and visit
   http://127.0.0.1:8000/
